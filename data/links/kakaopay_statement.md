# 카카오페이 거래내역서 발급

## 발급 방법
카카오페이 챗봇을 통해 거래내역서 발급 가능

## 접근 방법
1. 카카오톡 실행
2. 카카오페이 채팅방 접속
3. 챗봇에게 "거래내역서" 또는 "거래내역" 요청
4. 기간 선택 후 발급

## 메모
- 이메일로 거래내역서 수신 가능
- PDF 형식으로 제공
- 모바일에서 간편하게 발급 가능
- 카카오페이 앱 또는 카카오톡에서 이용 가능

## 참고
- 웹 페이지보다 챗봇 이용이 더 편리함
- 사업자용 거래내역도 발급 가능
